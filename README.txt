# Dev Metrics

Metricas mensuales de tickets por dev.

## Uso y buenas prácticas

El programa agarra los excels que usamos para medir los datos, filtra la información que necesitamos, y la larga en un nuevo excel que luego podemos guardar como pdf, y presentarlo.

No se debe cambiar el nombre de los archivos, porque sino el programa no los va a reconocer.
